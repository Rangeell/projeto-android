## 🚧 Dificuldades encontradas

- Fazer o vídeo se adaptar corretamente a diferentes tamanhos de tela
- Entender como o padding do container influenciava no layout
- Posicionar elementos sem quebrar o design
- Trabalhar com listas lado a lado
- Resolver problemas de responsividade em dispositivos menores


### 🔧 Como resolvi

- Utilizei a técnica de proporção com padding-bottom para tornar o vídeo responsivo
- Usei position: relative e position: absolute para controlar o posicionamento do iframe
- Apliquei margin negativo para ajustar elementos dentro do layout
- Utilizei display: inline-block para alinhar listas lado a lado
- Testei diferentes soluções até encontrar a que melhor funcionava para cada problema
